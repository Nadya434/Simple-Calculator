# API Reference

## Функции калькулятора

### `add(a, b)`
Сложение двух чисел.
- Параметры: a (число), b (число)
- Возвращает: сумму a + b

### `subtract(a, b)`
Вычитание b из a.
- Параметры: a (число), b (число)
- Возвращает: разность a - b

### `multiply(a, b)`
Умножение.
- Параметры: a (число), b (число)
- Возвращает: произведение a * b

### `divide(a, b)`
